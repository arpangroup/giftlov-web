# Gitlov WebPos React(Vite) App
[![Build Status](https://vitejs.dev/)](https://github.com/arpangroup/giftlov-api)

## _Steps to run the App:_
```sh
cd ./giftlov-web
npm install
npm run dev
```

## _Backend API:_
[Java Backend](http://localhost:8080) Endpoint can be configure in src/api.js/
